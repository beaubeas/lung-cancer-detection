Here you'll find REST API endpoints to create, update, delete, and interact with skills and agents.

### REST API for Skills

Here is how to interact with skills using REST API endpoints:

* `GET /api/projects/<project_name>/skills` - gets all skills created by the user (allow params to filter by agent)
* `POST /api/projects/<project_name>/skills` - creates a new skill
* `GET /api/projects/<project_name>/skills/<skill_name>` - gets a skill by name or ID
* `PUT /api/projects/<project_name>/skills/<skill_name>` - updates skill with new params
* `DELETE /api/projects/<project_name>/skills/<skills>` - deletes a skill by name or ID

### REST API for Agents

Here is how to interact with agents using REST API endpoints:

* `GET /api/projects/{projectName}/agents` - gets all agents created by the user
* `POST /api/projects/{projectName}/agents` - creates a new agent

* `GET /api/projects/{projectName}/agents/{agentName}` - gets an agent by name or ID
* `PUT /api/projects/{projectName}/agents/{agentName}` updates agent with new settings
* `DELETE /api/projects/{projectName}/agents/{agentName}` - deletes an agent by name or ID

* `POST /api/projects/{projectName}/agents/{agentName}/completions` - query an agent

    Request body:

    ```
    {
        "messages": [
            { "question": "What is the meaning of life?", "answer": "The meaning of life is a philosophical question that no one knows." },
            { "question": "Why do you not understand?", "answer": null }
        ]
    }
    ```

    Response body:

    ```
    "message": {
        "role": "assistant",
        "content": "I'm AI!"
    }
    ```
